

# Libero UART Bare-Metal Project

This folder contains the complete Libero SoC project and bare-metal UART example.

## Contents
- Libero SoC project files  
- MSS configuration  
- UART driver files  
- Bare-metal firmware (C code)  
- Documentation & notes  

## Tools Used
- Microchip Libero SoC  
- SoftConsole (for firmware)  
- SmartFusion2 / IGLOO2 FPGA  

## How to Use
1. Open the Libero project inside this folder  
2. Regenerate the component flow  
3. Build the bitstream  
4. Import firmware into SoftConsole  
5. Program FPGA + MSS  
6. Open UART terminal and test
